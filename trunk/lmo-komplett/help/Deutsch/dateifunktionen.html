Dateifunktionnen

  1. Löschen einer Liga
  2. Upload einer Liga
  3. Download/Mailen einer Liga

1. L&#246;schen einer Liga

Ist eine Liga veraltet oder wollen Sie sie durch eine Sicherheitskopie ersetzen, w&#228;hlen Sie im Men&#252; den Befehl L&#246;schen. Klicken Sie in der Ligenauflistung einfach die zu l&#246;schende Liga an. Es erfolgt eine Sicherheitsabfrage, so da&#223; Sie einen falschen Klick nochmal zur&#252;cknehmen k&#246;nnen. Wenn Sie sie aber best&#228;tigen, wird die Liga unwiderruflich gel&#246;scht. Seien Sie also vorsichtig mit der L&#246;schfunktion![nach oben]

2. Upload einer Liga

W&#228;hlen Sie den Men&#252;-Befehl Upload, um eine Liga von Ihrer lokalen Festplatte auf Ihren Server direkt in Ihr Ligenverzeichnis zu &#252;bertragen. Klicken Sie auf Durschsuchen..., um die Liga auf Ihrer Festplatte auszuw&#228;hlen. W&#228;hlen Sie bitte nur LM98Free/LMO-Ligen (*.L98) aus! Mit einem Klick auf Upload starten beginnt die &#220;bertragung (bitte nur einmal anklicken!).

Bereits auf dem Server befindliche Ligen werden mit dieser Funktion nicht &#252;berschrieben. Gibt es bereits eine Liga mit dem gleichen Dateinamen, wird die neue Datei nummeriert; aus liga.l98 wird dann liga_1.l98.[nach oben]

3. Download/Mailen einer Liga

M&#246;chten Sie eine Sicherheitskopie einer Liga anlegen, eine Liga offline weiterbearbeiten oder sie an einen Bekannten schicken, w&#228;hlen Sie aus dem Men&#252; den Befehl Download.

Der Downloadbildschirm enth&#228;lt die Auflistung aller Ligen zweimal. Im linken Bereich klicken Sie auf eine Liga, um sie auf Ihre lokale Festplatte runterzuladen. Klicken Sie auf Alle Ligen auf einmal downloaden, stellt Ihnen das Skript eine ZIP-Datei zusammen, in der es alle aufgelisteten Ligen archiviert, und bietet Ihnen dann dieses Archiv zum Download an.

Im rechten Bereich klicken Sie die Ligen an, um sie per E-Mail zu versenden. Bei einem Klick &#246;ffnet sich zuerst ein Eingabefenster, in dem Sie die E-Mail-Adresse des Empf&#228;ngers eingeben k&#246;nnen. Jetzt wird die gew&#228;hlte Liga bzw. alle Ligen zuerst in eine ZIP-Datei gepackt, die dann per E-Mail versandt wird. Ihre E-Mail-Adresse, wie Sie sie bei den Optionen eingegeben haben, wird als Absender der E-Mail angegeben.[nach oben]
