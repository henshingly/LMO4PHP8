1. Einleitung

Herzlich Willkommen zum neuen Liga Manager Online. Mit diesem Skript k&#246;nnen Sie Ihren Besuchern einen Ergebnisdienst &#252;ber Ihre Lieblingsligen anbieten. Die gesamte Verwaltung der Ligen l&#228;uft dabei direkt auf dem Server ab; Sie tragen nur noch die Ergebnisse ein und Ihre Besucher erhalten sofort die aktuellen Tabellen, Statistiken usw. - Keine Fremdsoftware, keine zeitraubenden Exporte und Uploads mehr!

Es ist ratsam, wenn Sie sich ein Bookmark/Favoriten in Ihrem Browser f&#252;r den Adminbereich (.../lmoadmin.php) einrichten, damit Sie immer schnell darauf zugreifen k&#246;nnen. Im User-Bereich ist aus Sicherheitsgr&#252;nden kein Link auf den Adminbereich vorhanden (damit niemand auf dumme Gedanken kommt).

Die Besucher k&#246;nnen das Skript &#252;ber die Datei lmo.php starten. Wie man den LMO an seine eigene Homepage anpasst, steht im Abschnitt Anpassung des LMO. Zu jedem Link erhalten Ihre Besucher einen Tooltip, so da&#223; Sie keine weiteren Erkl&#228;rungen zur Benutzung des Skripts geben m&#252;ssen.

Der Adminbereich ist, &#252;ber lmoadmin.php erreichbar. Beim ersten Start melden Sie sich mit dem Namen admin und dem Passwort lmo (Mit L wie Ludwig!) an. Unter Optionen k&#246;nnen und sollten Sie das dann sofort &#228;ndern, damit kein Unbefugter mehr Zugriff hat.[nach oben]

3. Installation

  1. Das ZIP-Archiv wurde bereits von Ihnen entpackt. Achten Sie darauf, dass die Verzeichnisstruktur beim Entpacken beibehalten wird.
  2. Falls Sie den LMO auf einem Linuxserver installieren, &#228;ndern Sie die Zugriffsrechte der Datei init-parameters.php auf 666 - wie das geht, verr&#228;t die Bedienungsanleitung Ihres FTP-Programms.
  3. Danach rufen Sie das Installationsskript install.php, welches sich im Verzeichnis lmo befindet und folgen den Anweisungen.
  4. Im Normalfall ist der LMO nun korrekt installiert. Sollte jedoch ein Fehler aufgetreten sein oder Sie k&#246;nnen aus anderen Gr&#252;nden die Installation nicht abschlie&#223;en, m&#252;ssen Sie die folgenden Schritte manuell nachvollziehen:

    Manuelle Installation

    • &#196;ndern Sie die Datei init-parameters.php mit einem normalen Texteditor und tragen Sie dort die ben&#246;tigten Pfade ein. Dies sind zum Ersten die URL und zum Anderen der Pfad zum LMO-Verzeichnis, falls Sie diese Angaben nicht wissen, fragen Sie bitte Ihren Provider.
    • Nun &#228;ndern Sie bei folgenden Datei die Zugriffsrechte (chmod):
      Verzeichnisse:
      777 - das Verzeichnis ligen und dessen Unterverzeichnis archiv
      777 - das Verzeichnis output
      777 - das Verzeichnis addon/spieler/stats
      777 - das Verzeichnis addon/tipp/tipps und dessen Unterverzeichnisse einsicht, auswert und vereine
      Dateien:
      666 - lmo-auth.txt
      666 - addon/tipp/lmo-tippauth.txt
      666 - Alle Dateien cfg.txt im Verzeichnis config und dessen Unterverzeichnissen
      666 - alle Ligen im ligen-Verzeichnis und im Unterverzeichnis archiv
      666 - alle Dateien in den Verzeichnissen output , addon/spieler/stats und tipp/tipps (sofern vorhanden)
[nach oben]

4. Login/Logout

Wenn Sie den Adminbereich betreten, sehen Sie zuerst eine Eingabemaske, mit der Sie sich beim System anmelden m&#252;ssen. Nach der Installation ist der voreingestellte der Nutzername admin und das Passwort lmo (kleines L, kleines M, kleines O) eingestellt. Das sollten Sie aber schnellstm&#246;glich &#228;ndern, um Unbefugten den Zutritt zu Ihrem Adminbereich zu verwehren.

Wenn Sie eingeloggt sind, haben Sie rechts oben im Men&#252; immer auch einen Befehl Logout, mit dem Sie sich aus dem Adminbereich ausloggen k&#246;nnen. Das sollten Sie der Sicherheit wegen auch immer am Ende einer Sitzung tun.[nach oben]
