047=This script based on the original Windows- Software "LigaManager '98 Free" by Frank Hollwitz. The <strong>Liga Manager Online</strong>, which is also developed by Frank Hollwitz, is an improved version of this software. Due to the fact that it is a PHP based software, the LMO is usable on many different OS with a browser for the GUI. With a lot of 3rd party addons the LMO grow up to a all-round application. In case of Frank Hollwitz didn't continue to develop this script, a group of 3rd party coder proceed to improve and to upgrade the LMO.
048=Man kann mit diesem Skript sowohl <strong>normale Ligen</strong> verschiedener Sportarten, als auch <strong>Pokalwettbewerbe</strong> verwalten, weiterhin sind unter anderem ein <strong>bet game</strong>, eine <strong>Spielerverwaltung</strong>, die Unterstützung für Mobile Geräte (<strong>WAP</strong>) sowie umfangreiche <strong>Ligastatistiken</strong> integriert. Somit ist der LMO <strong>die Komplettlösung</strong> für alle, die Spielergebnisse von Sportligen verwalten und öffentlich zugänglich machen wollen.
049=Das Script läuft browserunabhängig, da es serverseitig arbeitet. Ihr Browser sollte aber CSS unterstützen, um eine optimale Darstellung zu gewährleisten. Wenn der Webmaster den eingebauten Newsticker nutzt, wird dieser nur bei aktiviertem JavaScript angezeigt.
